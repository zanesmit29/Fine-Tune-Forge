Backend changes:
1. Add POST /integrations/modal endpoint:
   - Accepts token_id and token_secret in request body
   - Validates credentials by making a lightweight 
     Modal API test call
   - On success: store as session-scoped environment 
     variables (NOT in SQLite — keep in memory only 
     for security)
   - Returns { connected: true, verified_at: timestamp }
2. Add DELETE /integrations/modal endpoint to 
   disconnect (clears the session variables)
3. Add GET /integrations/modal/status endpoint 
   returning connection status + masked token ID
4. Update the /train endpoint:
   - If compute_mode is "gpu": check Modal credentials 
     are present in session, return 401 with message 
     "Connect your Modal account in Integrations to 
     use GPU training" if missing
   - Pass user's token credentials to modal_trainer.py

- If Modal is connected: show the existing
